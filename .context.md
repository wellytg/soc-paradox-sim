# .context.md: SOC Strategic Alignment Simulator (soc-paradox-sim)

## Domain Definition
The **SOC Strategic Alignment Simulator** is an interactive, research-oriented single-file web application designed to demonstrate the "Speed-Quality Paradox" in Security Operations Centers. It serves as the core data collection tool for the SOC Strategic Alignment study.

## Technical Architecture
- **Frontend:** Vanilla JavaScript, Tailwind CSS, HTML5.
- **Backend:** Firebase (Authentication & Firestore) for real-time global leaderboard synchronization and data logging.
- **Visual Mechanics:** Dynamic shape generation (Triangles vs. Distractors) to test analyst cognitive load.

## Strategic Objectives
- **Demonstrate Trade-offs:** Force analysts to choose between Rapid Triage (high volume) and Deep Review (high precision).
- **Log Performance Metrics:** Capture alert volume, detection success (Threat Code), and strategy coherence.
- **Pedagogical Feedback:** Provide instant analysis based on the "Productivity Trap" and "Strategic Alignment" frameworks.
